// Quiz

What is the difference between var and let?
    var is only block scoped to a function otherwise it's globally scoped
    let is block scoped to whatever it is inside of
What is the difference between var and const?
    var is only block scoped to a function otherwise it's globally scoped
    while const is block scoped
What is the difference between let and const?
    let can be reassigned
    const cannot be reassigned
What is hoisting?
    hoisting is what happens to var declared variables which are initalized before the rest of the JS code
    the variable you have set will be set within your code after the variable is originally declared
